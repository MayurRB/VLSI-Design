# HDLBits - Combinational Logic

## Problems Covered
- AND Gate
- Vector Operations
- 7458 Logic Circuit

## Concepts Used
- Continuous assignment (assign)
- Bitwise operations
- Multi-level combinational logic

##  Description

### 1. AND Gate
Implements basic AND logic using assign statement.

### 2. Vector Example
Demonstrates bitwise operations on 4-bit inputs.

### 3. 7458 Logic
Implements a combinational circuit using AND-OR logic.

## Learning Outcome
- Understanding how Verilog maps to hardware
- Writing clean combinational logic
- Working with vectors and expressions
